Scoping & context

Every conversation is anchored to a context — the slice of the grid the analyst is asking about. Context is captured as removable chips in a bar at the top of the Chats view, set by clicking cells in the grid, and mirrored back as colored highlighting on the grid itself. With nothing in focus, Scout answers about the whole view.

Context chips

The context bar holds one or more chips describing the current scope. Three kinds exist:

  • Row chip — a whole row (one customer / location / product line).
  • Cell chip — one specific value, such as a single proposed price.
  • Selection chip — a multi-cell pick made within the scoping rules.

When empty, the bar shows the hint Nothing in focus · click a cell alongside a + Add a cell affordance (see 1, 2). When populated, each chip is a removable pill.

Default click behavior

While the panel is open, clicking a grid cell replaces the current non-pinned context with a fresh chip — a cell chip for a data column, a row chip for an identity column. Once any chip is explicitly pinned, further clicks append instead of replace. Clicking a chip's own target toggles that chip off. With the panel open, Scout's scoping intercepts the grid's normal click so a click scopes Scout rather than opening the row's drawer.

Click interception Interception applies only while the panel is open. When Scout is closed, grid clicks behave normally. A rebuild must scope Scout without breaking the grid's own row interactions.

+Add mode and the no-diagonal rule

The + Add a cell affordance enters a focused picking mode: the cursor becomes a copy cursor and eligible cells gain a dashed outline. A multi-cell pick must share one row or one scope-group column — never a diagonal. Off-axis cells and identity cells are disabled while picking (see 1, 2). Pressing Esc exits the mode.

Grid cell highlighting

Highlighting appears only while Scout is open, and it encodes scope at a glance:

TreatmentMeaning
Soft green tint, inset outline, accented textScope cells — what the current context covers
Faint grey overlayIdentity cells — the row's descriptive columns
Subtle background tint per groupColumn-group tints — Cost, Proposed, Margin, Δ-vs-prior, Flags
Violet outline badgeAn explicit cell chip, marking its exact cell
