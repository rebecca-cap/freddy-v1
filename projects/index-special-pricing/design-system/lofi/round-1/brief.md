# Round 1 brief

What this round resolves (from the PM). Seeded from the 2026-06-15 sync — see
`./context/insights.md`.

## Focus

- A **single-view Quick Create for an index special offer** — fresh lo-fi, not the old drawer.
- Make the **index price formula** (index + differential, invoiced at lift time) the visual and
  interaction centerpiece, and fast to set.
- **Template-first** speed path, with **create-from-prior** prepopulation intersecting the same
  view (tweak, don't author).
- **Timing as durations** (visibility + pickup) rather than explicit days/dates.
- **Customers via existing groups / group-tags** (e.g. Gulf Coast); product = Group 3 gas.
- Demonstrate the **click-count / time budget**: ~10–12 essential clicks, ≤ 30 seconds.
- Show how the **same field sequence reads on mobile**.
- Power-user affordances: keyboard-first, hotkeys, templates.

## Out of scope

- **No AI** assistance in this design (explicit).
- Full prescriptive no-template authoring flow (only referenced as the fallback, not designed in
  depth this round).
- Hi-fi visuals / production styling / the OSP protobase refresh (separate groundwork).
- Backend wiring, real index data feeds, contract execution beyond "offer sent."
