Usage Guidelines

Filter Bar - Use for complex filtering scenarios with multiple filter options
Active Filters Pills - Show applied filters as removable pills for easy management
Clear All - Always provide a way to reset all filters at once
Apply Button - Use explicit apply when filters should not auto-apply
