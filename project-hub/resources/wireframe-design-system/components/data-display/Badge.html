Usage Guidelines

Default (Gray) - Use for neutral information, categories, or inactive states
Primary (Blue) - Use for informational states, in-progress items, or highlighting
Success (Green) - Use for positive states: active, completed, approved, online
Warning (Yellow/Orange) - Use for attention-needed states: pending, review required
Error (Red) - Use for negative states: failed, expired, errors, offline
