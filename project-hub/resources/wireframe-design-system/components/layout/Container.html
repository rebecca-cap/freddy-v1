Container Component

The Container component provides horizontal padding and optional max-width constraints for centering content. Use it to create consistent content widths across your layouts.

Basic Container

Full-width container with horizontal padding

<div class="wf-container">
  <!-- Your content here -->
</div>
Container with default padding (16px on each side)
width: 100% | padding: 16px horizontal

Container Size Variants

Constrain content width with max-width modifiers

<div class="wf-container wf-container-sm">...</div>  <!-- max-width: 640px -->
<div class="wf-container wf-container-md">...</div>  <!-- max-width: 768px -->
<div class="wf-container wf-container-lg">...</div>  <!-- max-width: 1024px -->
<div class="wf-container wf-container-xl">...</div>  <!-- max-width: 1280px -->
<div class="wf-container wf-container-2xl">...</div> <!-- max-width: 1536px -->

Container Padding Variants

Adjust horizontal padding with modifier classes

<div class="wf-container wf-container-tight">...</div>    <!-- padding: 8px -->
<div class="wf-container">...</div>                        <!-- padding: 16px (default) -->
<div class="wf-container wf-container-relaxed">...</div>  <!-- padding: 24px -->
<div class="wf-container wf-container-spacious">...</div> <!-- padding: 32px -->
.wf-container-tight (8px padding)
Default container (16px padding)
.wf-container-relaxed (24px padding)
.wf-container-spacious (32px padding)

Nested Containers

Containers can be nested for complex layouts

<div class="wf-container wf-container-xl">
  <div class="wf-container wf-container-md">
    <!-- Narrower content within wider container -->
  </div>
</div>

Outer: .wf-container-xl

Inner: .wf-container-md

Combined with Layout Components

Use containers with rows and columns for complete layouts

<div class="wf-container wf-container-lg">
  <div class="wf-row wf-gap-6">
    <aside class="wf-column wf-gap-4" style="width: 200px;">
      <!-- Sidebar -->
    </aside>
    <main class="wf-column wf-gap-4 wf-flex-1">
      <!-- Main content -->
    </main>
  </div>
</div>
