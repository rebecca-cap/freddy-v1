Row Component

The Row component creates a horizontal flexbox container for laying out items in a row. Combine with alignment, justification, and gap utilities for full control.

Basic Row

Items arranged horizontally with default alignment

<div class="wf-row">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

Combined Example: Header Layout

Common pattern: logo on left, nav in center, actions on right

    <a href="#">Home</a>
