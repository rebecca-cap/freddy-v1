Page Component

The Page component provides a full-page layout structure with a header (title + actions) and main content area. It establishes the primary visual hierarchy for application pages.

Basic Page Structure

Standard page layout with header and content area

<div class="wf-page">
  <header class="wf-page-header">
    <div class="wf-page-title">
      <h1 class="wf-text-h1">Page Title</h1>
      <p class="wf-text-helper">Optional subtitle or description</p>
    </div>
    <div class="wf-page-actions">
      <button>Secondary</button>
      <button>Primary Action</button>
    </div>
  </header>
  <main class="wf-page-content">
    <!-- Page content here -->
  </main>
</div>

CSS Class Reference

Class Description
.wf-page Full page container with min-height: 100vh, flex column layout
.wf-page-header Page header with title and actions, white background
.wf-page-title Container for h1 title and optional subtitle
.wf-page-actions Container for action buttons in header
.wf-page-content Main content area with flex-grow and padding
.wf-page-content-tight Reduced padding (16px)
.wf-page-content-spacious Increased padding (32px)
.wf-page-content-none No padding (0px)
