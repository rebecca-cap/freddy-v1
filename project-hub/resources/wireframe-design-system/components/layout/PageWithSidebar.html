Page with Sidebar

A two-column layout with a fixed sidebar for navigation and a flexible main content area. Common pattern for admin dashboards and application interfaces.

Basic Structure

Two-column layout with sidebar and main content area

<div class="wf-page wf-page-with-sidebar">
  <aside class="wf-sidebar">
    <div class="wf-sidebar-header">
      <!-- Logo/Brand -->
    </div>
    <nav class="wf-sidebar-nav">
      <!-- Navigation items -->
    </nav>
    <div class="wf-sidebar-footer">
      <!-- User info/settings -->
    </div>
  </aside>
  <main class="wf-main">
    <header class="wf-page-header">
      <!-- Page title and actions -->
    </header>
    <div class="wf-page-content">
      <!-- Page content -->
    </div>
  </main>
</div>

CSS Class Reference

Class Description
.wf-page-with-sidebar Modifier for .wf-page, changes to flex-direction: row
.wf-sidebar Fixed-width sidebar (256px), full height, white background
.wf-sidebar-collapsed Narrow sidebar (64px) for icon-only navigation
.wf-sidebar-header Logo/brand area at top of sidebar
.wf-sidebar-nav Scrollable navigation area (flex: 1)
.wf-sidebar-footer User info/settings at bottom of sidebar
.wf-main Main content wrapper (flex: 1, column layout)
