Layout Components

Pure HTML/CSS layout primitives for building wireframe mockups. No JavaScript required.

Quick Start

Include the CSS files in your HTML to start using the layout components:

<!-- Include design tokens and layout CSS -->
<link rel="stylesheet" href="tokens/index.css">
<link rel="stylesheet" href="components/layout/layout.css">

<!-- Basic page structure -->
<div class="wf-page">
  <header class="wf-page-header">
    <div class="wf-page-title">
      <h1 class="wf-text-h1">My Page</h1>
    </div>
  </header>
  <main class="wf-page-content">
    <div class="wf-row wf-gap-4">
      <div class="wf-card wf-flex-1">Card 1</div>
      <div class="wf-card wf-flex-1">Card 2</div>
    </div>
  </main>
</div>
