Column Component

The Column component creates a vertical flexbox container for stacking items vertically. Uses the same alignment utilities as Row but with axes swapped.

Basic Column

Items stacked vertically with default alignment

<div class="wf-column">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

Flex Grow

Use .wf-flex-1 or .wf-grow to make items fill available space

<div class="wf-column" style="height: 400px;">
  <header>Header (fixed)</header>
  <main class="wf-flex-1">Content (grows)</main>
  <footer>Footer (fixed)</footer>
</div>
