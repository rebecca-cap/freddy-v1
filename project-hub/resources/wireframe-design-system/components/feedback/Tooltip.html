Tooltip Components

Contextual information that appears on hover. Useful for providing hints or additional details.

Tooltip Positions

Tooltips can appear above, below, to the left, or to the right of the trigger element. Hover over the buttons to see the tooltips.

Usage

Copy and paste these patterns into your wireframes.

<!-- Basic Tooltip (shows on hover) -->
<div class="wf-tooltip-container">
  <button class="wf-button wf-button-secondary">Hover me</button>
  <div class="wf-tooltip wf-tooltip-top">
    This is a tooltip
    <span class="wf-tooltip-arrow"></span>
  </div>
</div>

<!-- Always Visible (for wireframes) -->
<div class="wf-tooltip-container">
  <button class="wf-button wf-button-secondary">Button</button>
  <div class="wf-tooltip wf-tooltip-top wf-tooltip-visible">
    Always shown
    <span class="wf-tooltip-arrow"></span>
  </div>
</div>

<!-- Multi-line Tooltip -->
<div class="wf-tooltip-container">
  <span>Info</span>
  <div class="wf-tooltip wf-tooltip-bottom wf-tooltip-multiline">
    This tooltip has a longer description that wraps to multiple lines.
    <span class="wf-tooltip-arrow"></span>
  </div>
</div>
