Toast Components

Non-blocking notification messages that appear temporarily.

Toast Variants

Four semantic variants for different types of feedback: info, success, warning, and error.

ℹ Information message - something to be aware of.
✓ Changes saved successfully!
⚠ Warning: Your session will expire in 5 minutes.
✕ Error: Unable to save changes. Please try again.

Auto-dismiss Toasts

Toasts without a close button auto-dismiss after a timeout.

✓ File uploaded successfully
ℹ Syncing in progress...

Long Content

Toasts gracefully handle longer messages.

⚠ Your subscription will expire on January 31, 2026. Please update your payment method to avoid service interruption.

Toast Positions

Toasts can appear in different corners or centered at top/bottom.

Top Right (default)

✓ Saved

Top Left

ℹ Info

Bottom Right

⚠ Warning

Bottom Left

✕ Error

Top Center

✓ Centered

Bottom Center

ℹ Centered

Stacked Toasts

Multiple toasts stack vertically in the container.

✓ Draft saved
ℹ Uploading file...
✓ Email sent

Usage

Copy and paste these patterns into your wireframes.

<!-- Toast Container (position at bottom-right by default) -->
<div class="wf-toast-container">

  <!-- Info Toast -->
  <div class="wf-toast wf-toast-info">
    <span class="wf-toast-icon">&#8505;</span>
    <span class="wf-toast-message">Information message</span>
    <button class="wf-toast-close">&times;</button>
  </div>

  <!-- Success Toast -->
  <div class="wf-toast wf-toast-success">
    <span class="wf-toast-icon">&#10003;</span>
    <span class="wf-toast-message">Changes saved successfully</span>
  </div>

  <!-- Warning Toast -->
  <div class="wf-toast wf-toast-warning">
    <span class="wf-toast-icon">&#9888;</span>
    <span class="wf-toast-message">Warning message</span>
    <button class="wf-toast-close">&times;</button>
  </div>

  <!-- Error Toast -->
  <div class="wf-toast wf-toast-error">
    <span class="wf-toast-icon">&#10005;</span>
    <span class="wf-toast-message">Error message</span>
    <button class="wf-toast-close">&times;</button>
  </div>

</div>
