Confirm Dialog Components

Confirmation modals for destructive actions or important decisions. Uses the small modal size with centered content.

Confirm Dialog Variants

Different icon styles for various confirmation contexts: info, success, warning, and danger.

Danger - Delete Confirmation

⚠

Delete Contract?

This action cannot be undone. All associated data will be permanently removed.

Warning - Discard Changes

⚠

Discard Changes?

You have unsaved changes that will be lost if you leave this page.

Info - Send Notification

ℹ

Send Notification?

This will send an email notification to 24 team members.

Success - Publish Confirmation

✓

Ready to Publish?

Your article will be visible to all users once published.

Full Overlay Example

How the confirm dialog looks within a full overlay context.

🗑

Delete Account?

This will permanently delete your account and all associated data. This action cannot be undone.

Usage

Copy and paste these patterns into your wireframes.

<!-- Confirm Dialog (within overlay) -->
<div class="wf-overlay wf-overlay-visible">
  <div class="wf-overlay-backdrop"></div>
  <div class="wf-modal wf-modal-sm wf-modal-confirm">
    <div class="wf-modal-body">
      <div class="wf-confirm-icon wf-confirm-icon-warning">&#9888;</div>
      <h3 class="wf-confirm-title">Delete Contract?</h3>
      <p class="wf-confirm-message">This action cannot be undone.</p>
    </div>
    <div class="wf-modal-footer">
      <button class="wf-button wf-button-secondary">Cancel</button>
      <button class="wf-button wf-button-danger">Delete</button>
    </div>
  </div>
</div>

<!-- Icon Variants -->
<!-- wf-confirm-icon-info (blue background) -->
<!-- wf-confirm-icon-success (green background) -->
<!-- wf-confirm-icon-warning (yellow background) -->
<!-- wf-confirm-icon-danger / wf-confirm-icon-error (red background) -->

<!-- Loading State -->
<div class="wf-modal wf-modal-sm wf-modal-confirm">
  <div class="wf-modal-body">
    <div class="wf-loading">
      <div class="wf-loading-spinner"></div>
      <span class="wf-loading-text">Processing...</span>
    </div>
  </div>
</div>
