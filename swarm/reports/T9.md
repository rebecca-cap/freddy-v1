## Envelope shape

These endpoints do NOT use the `{ Data: ... }` envelope. The consumers (`ByCustomer/page.tsx`, `ByChannel/page.tsx`, `ByTerminal/page.tsx`, `ByContract/page.tsx`) read `data.GraphRows` and `data.GridRows` directly off the response object, so the fixture is the response object itself with `GraphRows`, `GridRows`, `HasData` at the top level.

- Date range: 2026-02-01 → 2026-05-02 (91 days, daily). Values deterministic via Mulberry32 PRNG seeded per series, so reloading produces identical fixtures.
- Volumes (`Liftings`): low-thousands to ~30k (fits brief's "thousands of gallons").
- Margins: 0.04–0.20 with random walk, clamped to 0.02–0.45.
- Mild seasonal sine overlay on volumes.
- Per-row `SystemAverageMetrics` populated on every graph point (consumers use it for the comparison band).
